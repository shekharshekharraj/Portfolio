# 💼 Raj Shekhar – Full Stack Developer Portfolio

Welcome to my personal portfolio website! This is a fully responsive and interactive web application built using modern web technologies to showcase my projects, skills, education, and contact information.

Live Demo: localhost:5000

## 🚀 Tech Stack

- **Frontend:** React, TypeScript, Tailwind CSS, Vite
- **Icons:** [Lucide React](https://lucide.dev/)
- **Animation:** Tailwind Animate
- **Deployment:** Vercel / Netlify / GitHub Pages

---

## ✨ Features

- 🔥 Smooth scroll navigation between sections
- 📁 Projects section to highlight live demos and GitHub repos
- 📊 Skills and tech stack representation
- 🎓 Education timeline
- 📬 Contact section with mail integration
- 🌙 Dark mode ready (via Tailwind’s `darkMode`)
- 📱 Fully responsive on all screen sizes
